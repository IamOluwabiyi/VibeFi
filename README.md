# VibeFi

A savings app combining Ajo (rotating group savings) and locked goal-based savings, powered by the Monnify Sandbox API.

Built for APIConf Lagos 2026 x Monnify Developer Challenge.

## Features
- **Ajo Groups** — create or join rotating savings groups with friends
- **Locked Savings** — set a goal and target date; funds are locked until that date
- **Savings Suggestion** — AI suggests a realistic savings target based on your contribution history
- **Chat Assistant** — ask questions about your savings, goals, and Ajo schedule in plain language

## Built With
- [Your frontend framework, e.g. React]
- Monnify Sandbox API (payments, reserved accounts, disbursements)

## Setup Instructions
1. Clone this repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and fill in your own Monnify sandbox credentials (get yours at monnify.com under Developer > API Keys & Contracts)
4. Run the app: `npm start`
